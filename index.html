LABORATÓRIO

BIBLIOTECA

O Sistema de Biblioteca é uma aplicação web desenvolvida para gerenciar uma coleção de livros . Projetado com uma interface amigável, o sistema permite aos usuários adicionar novos livros à biblioteca, fornecendo informações detalhadas, como nome do livro e matéria relacionada.

GERENCIADOR CAVALINHO

Um Gerenciador de Rotas do Google Maps, onde os usuários podem cadastrar, visualizar, buscar e exportar rotas para o Google Maps. Também podemos gerenciar imagens, permitindo ao usuário adicionar, remover, copiar e buscar imagens armazenadas

COMPACTAÇÃO

O Compactador de Sequências é uma pequena aplicação JavaScript que utiliza um algoritmo simples para compactar sequências de caracteres.

PÁGINA DO GRÊMIO

A Página do Grêmio foi desenvolvida em HTML, CSS e JavaScript, oferece uma experiência interativa e informativa aos torcedores e visitantes.

PLAYLIST DO YOUTUBE

A Página de Playlists do YouTube oferece aos usuários a possibilidade de escutar suas playlists com a tela do celular desligada. Desenvolvida em HTML, CSS e JavaScript, a página proporciona uma interface simples e interativa.

ACADEMIA

O sistema de Academia contém uma aplicação web relacionada a treinos de academia, onde o usuário pode ajustar o número de séries, repetições e peso para diferentes exercícios.

JOGO DA VELHA

O Jogo da Velha é uma aplicação web que apresenta uma implementação básica e intuitiva deste tradicional passatempo. Desenvolvido utilizando HTML, CSS e JavaScript, o projeto proporciona uma experiência de usuário descomplicada e acessível.
