BIBLIOTECA

O Sistema de Biblioteca é uma aplicação web desenvolvida para gerenciar uma coleção de livros. Projetado com uma interface amigável, o sistema permite aos usuários adicionar novos livros à biblioteca, fornecendo informações detalhadas, como nome do livro e matéria relacionada.

Principais Funcionalidades:

Adição de Livros: Os usuários podem facilmente adicionar novos livros à biblioteca, incluindo detalhes como nome do livro, matéria correspondente e associando arquivos PDF e imagens PNG para fornecer conteúdo visual.

Visualização de Livros: A aplicação oferece uma visualização organizada da lista de livros existentes, destacando informações essenciais, como nome, matéria e capa do livro.

Abrir PDF: Uma funcionalidade integrada permite a visualização rápida dos conteúdos dos livros em formato PDF. Os usuários podem acessar o PDF associado a um livro com um simples clique.

Ordenação e Filtragem: Os livros podem ser organizados de forma eficiente por nome e matéria, proporcionando aos usuários a capacidade de personalizar a exibição da lista conforme suas preferências.

Exclusão de Livros: O sistema oferece a funcionalidade de excluir livros da biblioteca, proporcionando flexibilidade na gestão da coleção.

Desenvolvido com HTML, CSS e JavaScript, e fazendo uso do IndexedDB para armazenamento local, o Sistema de Biblioteca é uma solução prática e eficaz para quem deseja manter um controle eficiente de seus recursos literários. Seu design modular facilita a expansão e a personalização para atender às necessidades específicas de cada usuário.
